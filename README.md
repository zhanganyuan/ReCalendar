# ReCalendar
爬取武大教务系统，获取成绩、课程，生成将课程一键导入系统日历的文件。
## 需先安装Python环境

## 打开命令行，安装一下包

1. `pip install BeautifulSoup`

2. `pip install icalendar`

3. `pip install urllib`

4. 如果提示还缺哪些包的话，就pip install 就行

## cd 到当前目录并运行程序

`python main.py`
